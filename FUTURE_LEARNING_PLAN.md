# Node.js Future Learning Plan

This document outlines the roadmap for upcoming learning sessions, tailored to advanced Node.js/React topics.

## Upcoming Learning

### 18. Real-Time Communication (WebSockets)
*   **Goal**: Building a live chat or notification system using `socket.io`.
*   **Why**: Understand Node.js's event-driven architecture for real-time apps, distinct from standard HTTP.

### 19. GraphQL
*   **Goal**: Exposing a GraphQL API in NestJS and consuming it with Apollo Client.
*   **Why**: Learn the "Code-First" approach in NestJS and how it differs from REST.

### 20. Server-Side Rendering (Next.js)
*   **Goal**: Introduction to Next.js concepts (SSR, SSG, Hydration).
*   **Why**: The industry standard for production React apps, bridging backend and frontend concepts.

### 21. End-to-End Testing (Playwright)
*   **Goal**: Automated browser testing for our full application.
*   **Why**: Completing the testing pyramid (Unit -> Integration -> E2E).

### 22. Behavior Driven Development (Cucumber)
*   **Goal**: Writing Gherkin features (`.feature` files) and automating them with simple steps.
*   **Why**: Learn to collaborate with non-technical stakeholders using the language of the business.
