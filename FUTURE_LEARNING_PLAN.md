# Node.js Future Learning Plan

This document outlines the roadmap for upcoming learning sessions. We will tackle these topics one by one.

## Completed Phases
- **Phase 1: Fundamentals** (Async/Await, Events, Modules)
- **Phase 2: HTTP & Config** (HttpModule, ConfigModule)
- **Phase 3: Production Readiness** (Input Validation, Global Error Handling)

## Phase 4: Frontend Development (React)

### 10. React Setup & Tailwind
*   **Goal**: Initialize a Vite project with specific versions (React 18.2, Vite 4.4.5).
*   **Tech**: Vite, Tailwind CSS.

### 11. Navigation & Routing
*   **Goal**: Client-side routing for a multi-page feel.
*   **Tech**: React Router DOM (v6.1.6).

### 12. State Management
*   **Goal**: Managing global application state.
*   **Tech**: Redux Toolkit, React Redux.

### 13. Micro-Frontends
*   **Goal**: Splitting the UI into host and remote applications.
*   **Tech**: `@originjs/vite-plugin-federation`.

### 15. Component Testing
*   **Goal**: Unit testing components with a visual UI.
*   **Tech**: **Vitest**, **@vitest/ui**, React Testing Library.

### 16. Integration
*   **Goal**: Connecting the React UI to our NestJS Backend.

## Next Steps (Proposals)
*   **Middleware**: Logging, Request ID tracking.
*   **Database**: Connecting to a real DB (TypeORM or Prisma).
*   **Authentication**: Real-world JWT implementation.
