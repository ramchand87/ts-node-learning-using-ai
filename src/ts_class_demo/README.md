# TypeScript Class & Module Demo

This tutorial demonstrates how to define classes and export/import modules using **TypeScript** and **ES Modules** syntax (`import` and `export`).

## What is Covered?

1.  **Strong Typing**: Defining types for class properties (`name: string`, `age: number`).
2.  **ES Exports**: Using `export class Person` to expose the class.
3.  **ES Imports**: Using `import { Person } from './Person'` to use the class.
4.  **Type Checking**: How TypeScript ensures you pass the correct data types to the constructor.

## Key Files

*   `Person.ts`: The typed class definition.
*   `index.ts`: The script that imports and uses the class.

## How to Run

```bash
npx ts-node src/ts_class_demo/index.ts
```
