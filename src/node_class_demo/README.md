# Node.js Class & Module Demo (CommonJS)

This tutorial demonstrates how to define classes and export/import modules using the standard Node.js **CommonJS** system (`require` and `module.exports`).

## What is Covered?

1.  **Class Definition**: Creating a simple `Person` class with a constructor and methods.
2.  **Module Exports**: Using `module.exports` to make the class available to other files.
3.  **Module Imports**: Using `require('./path/to/file')` to import the class.
4.  **Instantiation**: Creating objects from the imported class.

## Key Files

*   `Person.js`: The class definition.
*   `index.js`: The script that imports and uses the class.

## How to Run

```bash
node src/node_class_demo/index.js
```
